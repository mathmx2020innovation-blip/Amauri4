## 🚀 About this project

This repository contains the source code for my **professional interactive CV**.  
It's a modern, single-page web application that presents my profile in a bold, clean, and dynamic way — using a **grey / red / black** color palette and **modal windows** for each section.

### ✨ Key features

- **6 independent sections** (About, Experience, Education, Skills, Certifications, Achievements)
- Each section opens in a **modal window** — clean and focused user experience.
- Fully **responsive design** (works on desktop, tablet, and mobile).
- **English-first** presentation (ideal for international opportunities).
- Sticky navigation bar with hover effects.
- No external dependencies — pure HTML/CSS/JavaScript.

## 🎨 Design system

- **Background:** `#0c0d10` with subtle radial pattern
- **Primary accent:** `#c62828` / `#e53935` (red)
- **Cards & modals:** Dark surfaces (`#121318`, `#14161f`) with red left borders
- **Typography:** Inter (Google Fonts) — clean, modern, highly readable
- **Icons:** Font Awesome 6 (free tier)

---

## 🛠️ Technologies used

- HTML5
- CSS3 (Flexbox, Grid, animations, sticky positioning)
- Vanilla JavaScript (modal handling, keyboard events, click-outside-to-close)
- Font Awesome 6
- Google Fonts (Inter)
